## Get Your First Agent Payment Running in 5 Minutes

This guide walks you through creating an AI agent, provisioning a wallet, and executing your first policy-enforced payment.

<Steps>

<Step title="Get Your API Key">

Sign up at [sardis.sh](https://sardis.sh) and create an API key:

1. Go to **Settings** → **API Keys**
2. Click **Create API Key**
3. Copy your key — it starts with `sk_live_...` (production) or `sk_test_...` (sandbox)

<Warning>
  Keep your API key secret! Never commit it to version control. Use environment variables:
  ```bash
  export SARDIS_API_KEY=sk_live_...
  ```
</Warning>

</Step>

<Step title="Install the SDK">

<Tabs>
  <Tab title="Python">
    ```bash
    pip install sardis
    ```

    Requires Python 3.11+
  </Tab>
  <Tab title="TypeScript">
    ```bash
    npm install @sardis/sdk
    ```

    Requires Node.js 18+
  </Tab>
  <Tab title="MCP Server">
    No installation needed! MCP uses npx:

    ```json
    {
      "mcpServers": {
        "sardis": {
          "command": "npx",
          "args": ["-y", "@sardis/mcp-server"],
          "env": {
            "SARDIS_API_KEY": "sk_live_..."
          }
        }
      }
    }
    ```
  </Tab>
</Tabs>

</Step>

<Step title="Create an Agent and Wallet">

<CodeGroup>

```python Python
from sardis import SardisClient
from decimal import Decimal

# Initialize the client
sardis = SardisClient(api_key="sk_live_...")

# Create an AI agent
agent = sardis.agents.create(
    name="my-first-agent",
    description="My first Sardis agent",
)

print(f"Agent created: {agent.agent_id}")

# Create a wallet with spending policies
wallet = sardis.wallets.create(
    agent_id=agent.agent_id,
    chain="base",
    currency="USDC",
    limit_per_tx=Decimal("100.00"),  # Max $100 per transaction
    limit_total=Decimal("500.00"),   # Max $500 total
)

print(f"Wallet created: {wallet.wallet_id}")
print(f"Wallet address: {wallet.address}")
```


<Step title="Fund Your Wallet">

Before you can make payments, fund your wallet with USDC:

<Tabs>
  <Tab title="Test Mode (Sandbox)">
    Sandbox wallets come pre-funded with test USDC. No action needed!
  </Tab>
  <Tab title="Production Mode">
    Send USDC to your wallet address from:

    - Coinbase
    - Circle account
    - Another wallet
    - On-ramp service (Stripe, MoonPay)

    ```python
    # Check balance
    balance = sardis.wallets.get_balance(
        wallet.wallet_id,
        chain="base",
        token="USDC"
    )
    print(f"Balance: {balance.balance} USDC")
    ```
  </Tab>
</Tabs>

</Step>

<Step title="Execute Your First Payment">

Now execute a payment with automatic policy validation:

<CodeGroup>

```python Python
from sardis import SardisClient
from decimal import Decimal

sardis = SardisClient(api_key="sk_live_...")

# Execute a payment
result = sardis.wallets.transfer(
    wallet.wallet_id,
    destination="merchant@example.com",  # Or wallet address
    amount=Decimal("25.00"),
    token="USDC",
    chain="base",
    domain="openai.com",  # Merchant domain for policy checks
    memo="OpenAI API credits",
)

print(f"Payment status: {result.status}")
print(f"Transaction hash: {result.tx_hash}")
print(f"Amount: {result.amount} {result.token}")
print(f"From: {result.from_address}")
print(f"To: {result.to_address}")
print(f"Chain: {result.chain}")
```

<Tip>
  The payment is automatically validated against your wallet's spending policies:
  - ✅ Amount ($25) is under per-transaction limit ($100)
  - ✅ Total spending is under daily limit ($500)
  - ✅ Merchant domain is not blocked
</Tip>

## Complete Working Example

Here's a complete example from our [examples directory](https://github.com/EfeDurmaz16/sardis/tree/main/examples):

<CodeGroup>

```python examples/simple_payment.py
#!/usr/bin/env python3
"""
Simple Payment Demo
===================

This example shows how to create a wallet and execute a payment
using the Sardis payment protocol.
"""

from sardis import Wallet, Transaction

def main():
    # Create a wallet with initial balance
    print("1. Creating wallet with $50 USDC...")
    wallet = Wallet(initial_balance=50, currency="USDC")
    print(f"   Wallet ID: {wallet.wallet_id}")
    print(f"   Balance: ${wallet.balance} {wallet.currency}")
    
    # Create and execute a transaction
    print("2. Executing payment of $2 to OpenAI API...")
    tx = Transaction(
        from_wallet=wallet,
        to="openai:api",
        amount=2,
        purpose="GPT-4 API call"
    )
    
    result = tx.execute()
    print(f"   Transaction ID: {result.tx_id}")
    print(f"   Status: {result.status.value}")
    print(f"   TX Hash: {result.tx_hash}")
    
    # Check new balance
    print("3. Checking wallet balance...")
    print(f"   New Balance: ${wallet.balance} {wallet.currency}")
    print(f"   Total Spent: ${wallet.spent_total}")
    
    print("✓ Demo completed successfully!")

if __name__ == "__main__":
    main()
```

```typescript examples/vercel_ai_payment.ts
import { generateText, tool } from "ai";
import { openai } from "@ai-sdk/openai";
import { z } from "zod";
import { SardisClient } from "@sardis/sdk";

const sardis = new SardisClient({ apiKey: process.env.SARDIS_API_KEY });

async function main() {
  const agent = await sardis.agents.create({
    name: "vercel-ai-agent",
    description: "Vercel AI SDK procurement example agent",
  });

  const wallet = await sardis.wallets.create({
    agent_id: agent.agent_id,
    currency: "USDC",
    limit_per_tx: "100.00",
    limit_total: "500.00",
  });

  const { text } = await generateText({
    model: openai("gpt-4o"),
    system: "You are a procurement agent with a Sardis wallet.",
    prompt: "Purchase $25 of OpenAI API credits.",
    tools: {
      sardisPay: tool({
        description: "Execute a payment through Sardis",
        parameters: z.object({
          to: z.string(),
          amount: z.string(),
          token: z.enum(["USDC", "USDT", "EURC"]),
          purpose: z.string(),
        }),
        execute: async ({ to, amount, token, purpose }) => {
          const result = await sardis.wallets.transfer(wallet.wallet_id, {
            destination: to,
            amount,
            token,
            chain: "base",
            domain: to,
            memo: purpose,
          });
          return {
            status: result.status,
            txHash: result.tx_hash,
          };
        },
      }),
    },
  });

  console.log("Agent response:", text);
}

main();
```

## Common Issues

<AccordionGroup>
  <Accordion title="Payment failed: insufficient balance">
    Fund your wallet with USDC on the target chain (e.g., Base, Polygon). In sandbox mode, test wallets are pre-funded.
  </Accordion>
  <Accordion title="Payment failed: policy violation">
    Check your wallet's spending limits with `sardis.wallets.get(wallet_id)`. The payment may exceed per-transaction or daily limits.
  </Accordion>
  <Accordion title="Payment failed: blocked merchant">
    The merchant domain may be on your blocklist or flagged by compliance checks. Review your policy settings.
  </Accordion>
  <Accordion title="Transaction pending for too long">
    Blockchain transactions can take 1-5 minutes depending on network congestion. Check the block explorer for real-time status.
  </Accordion>
</AccordionGroup>
