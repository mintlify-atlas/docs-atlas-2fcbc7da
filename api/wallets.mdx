## The Wallet Object

A wallet is a non-custodial MPC wallet that holds stablecoins across multiple chains.

### Attributes

<ResponseField name="wallet_id" type="string">
  Unique wallet identifier
</ResponseField>

<ResponseField name="agent_id" type="string">
  Associated agent ID
</ResponseField>

<ResponseField name="mpc_provider" type="string">
  MPC provider: `turnkey`, `circle`, `fireblocks`, `local`
</ResponseField>

<ResponseField name="account_type" type="string">
  Account type: `mpc_v1` or `erc4337_v2`
</ResponseField>

<ResponseField name="addresses" type="object">
  Blockchain addresses by chain (e.g., `{"base": "0x123..."}`)
</ResponseField>

<ResponseField name="currency" type="string">
  Primary currency (e.g., `USDC`)
</ResponseField>

<ResponseField name="limit_per_tx" type="string">
  Per-transaction spending limit
</ResponseField>

<ResponseField name="limit_total" type="string">
  Total spending limit
</ResponseField>

## Create Wallet

```bash cURL
curl -X POST https://api.sardis.sh/api/v2/wallets \
  -H "Authorization: Bearer sk_live_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "agent_id": "agent_abc123",
    "mpc_provider": "turnkey",
    "currency": "USDC",
    "limit_per_tx": "100.00",
    "limit_total": "1000.00"
  }'
```

```python Python
wallet = client.wallets.create(
    agent_id="agent_abc123",
    mpc_provider="turnkey",
    currency="USDC",
    limit_per_tx=100.00,
    limit_total=1000.00
)
```

### Request Body

<ParamField body="agent_id" type="string" required>
  Agent to create wallet for
</ParamField>

<ParamField body="mpc_provider" type="string" default="circle">
  MPC provider: `circle`, `turnkey`, `fireblocks`
</ParamField>

<ParamField body="currency" type="string" default="USDC">
  Primary currency
</ParamField>

<ParamField body="limit_per_tx" type="number" default="100.00">
  Per-transaction limit
</ParamField>

<ParamField body="limit_total" type="number" default="1000.00">
  Total limit
</ParamField>

## Get Wallet Balance

```bash cURL
curl https://api.sardis.sh/api/v2/wallets/wallet_abc123/balance?chain=base&token=USDC \
  -H "Authorization: Bearer sk_live_your_api_key"
```

```python Python
balance = client.wallets.get_balance(
    wallet_id="wallet_abc123",
    chain="base",
    token="USDC"
)
print(f"Balance: {balance.balance} USDC")
```

### Response

```json
{
  "wallet_id": "wallet_abc123",
  "chain": "base",
  "token": "USDC",
  "balance": "1250.50",
  "address": "0x1234567890abcdef..."
}
```

## Transfer Crypto

```bash cURL
curl -X POST https://api.sardis.sh/api/v2/wallets/wallet_abc123/transfer \
  -H "Authorization: Bearer sk_live_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "destination": "0xRecipientAddress...",
    "amount": "50.00",
    "token": "USDC",
    "chain": "base",
    "memo": "Payment for API credits"
  }'
```

```python Python
tx = client.wallets.transfer(
    wallet_id="wallet_abc123",
    destination="0xRecipientAddress...",
    amount="50.00",
    token="USDC",
    chain="base",
    memo="Payment for API credits"
)
print(f"Transaction: {tx.tx_hash}")
```

### Request Body

<ParamField body="destination" type="string" required>
  Recipient wallet address (0x...)
</ParamField>

<ParamField body="amount" type="string" required>
  Amount to transfer (in token units)
</ParamField>

<ParamField body="token" type="string" default="USDC">
  Token to transfer
</ParamField>

<ParamField body="chain" type="string" default="base_sepolia">
  Blockchain to execute on
</ParamField>

<ParamField body="memo" type="string">
  Optional memo for audit trail
</ParamField>

### Response

```json
{
  "tx_hash": "0xabc123...",
  "status": "submitted",
  "from_address": "0x1234...",
  "to_address": "0x5678...",
  "amount": "50.00",
  "token": "USDC",
  "chain": "base",
  "fee_amount": "0.50",
  "net_amount": "49.50",
  "ledger_tx_id": "tx_ledger_123"
}
```

## Freeze Wallet

Freeze a wallet to block all transactions (for compliance or security).

```bash cURL
curl -X POST https://api.sardis.sh/api/v2/wallets/wallet_abc123/freeze \
  -H "Authorization: Bearer sk_live_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "reason": "Suspicious activity detected",
    "frozen_by": "admin@company.com"
  }'
```
