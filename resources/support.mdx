## Common Issues

### Transaction Failed

<Steps>
  <Step title="Check wallet balance">
    ```bash
    sardis wallets get <wallet_id>
    ```
  </Step>
  <Step title="Verify spending policy">
    ```python
    result = sardis.policies.check(
        wallet_id=wallet_id,
        amount="100",
        merchant="example.com"
    )
    if not result.allowed:
        print(f"Blocked: {result.reason}")
    ```
  </Step>
  <Step title="Check network status">
    Visit [status.sardis.sh](https://status.sardis.sh) for real-time status
  </Step>
  <Step title="Review error logs">
    Check the transaction error message:
    ```python
    print(f"Error: {result.error}")
    print(f"Code: {result.error_code}")
    ```
  </Step>
</Steps>

### MCP Server Not Working

<Steps>
  <Step title="Verify installation">
    ```bash
    npx @sardis/mcp-server@latest start
    ```
  </Step>
  <Step title="Check configuration">
    Ensure `claude_desktop_config.json` is correct:
    ```json
    {
      "mcpServers": {
        "sardis": {
          "command": "npx",
          "args": ["@sardis/mcp-server", "start"],
          "env": {
            "SARDIS_API_KEY": "sk_..."
          }
        }
      }
    }
    ```
  </Step>
  <Step title="Test API key">
    ```bash
    curl https://api.sardis.sh/v2/health \
      -H "Authorization: Bearer sk_..."
    ```
  </Step>
  <Step title="View logs">
    - Mac: `~/Library/Logs/Claude/mcp*.log`
    - Windows: `%APPDATA%\Claude\logs\mcp*.log`
  </Step>
</Steps>

## Security Reports

Found a security vulnerability? Please report it responsibly.

<Warning>
  **Do NOT create a public GitHub issue for security vulnerabilities.**
</Warning>

### Responsible Disclosure

<Steps>
  <Step title="Email security@sardis.sh">
    Include:
    - Vulnerability description
    - Steps to reproduce
    - Proof of concept (if applicable)
    - Your contact information
  </Step>
  <Step title="Wait for acknowledgment">
    We'll respond within 2 hours with next steps
  </Step>
  <Step title="Coordinate disclosure">
    We'll work with you on a fix and coordinated disclosure timeline
  </Step>
  <Step title="Receive recognition">
    Valid vulnerabilities are eligible for bug bounty rewards (up to $100k)
  </Step>
</Steps>
